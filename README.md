# plugin-splid

A Stavrobot plugin for managing shared expenses via [Splid](https://splid.app).

## What it does

This plugin lets you interact with your Splid expense-sharing groups through Stavrobot. You can:

- Check who owes whom and how much (**get_balance**)
- List recent expenses and payments (**get_expenses**)
- Add new expenses to the group (**add_expense**)

## Installation

Tell Stavrobot:

> Install the plugin at `https://github.com/skorokithakis/plugin-splid`

Then configure the invite code for your Splid group:

> Configure the splid plugin with the invite code `ABC DEF GHI`

You can find your group's invite code in the Splid app under group settings.
